# Jupiter, the Julia Compiler

Install environment:
`make install`

Run file:
`julia-1.0.0/bin/julia  "filename".jl`

Clean environment:
`make clean`

Para executar os testes sequencialmente chame o arquivo "testes.txt"
Caso queira rodar o fatorial, chame o arquivo "fatorial.txt"
